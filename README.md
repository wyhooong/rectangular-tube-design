# 矩形钢管截面缩尺自动化设计系统

基于相似理论的结构缩尺模型优化设计工具。

## 👉 [点击此处打开设计网站](https://rectangular-tube-design.netlify.app/)

所有计算在浏览器本地完成，无需安装任何软件，打开即用。

## 功能

- ⚡ 单工况优化设计（两阶段网格搜索）
- 📊 批量工况计算（最多50组）
- 📐 截面示意图实时绘制
- 📈 误差分布柱状图
- 📄 导出设计报告 / CSV
- 💾 方案历史记录

## 相似理论公式

| 性能指标 | 相似比 |
|---------|--------|
| 轴向刚度 | S_E · S_L² |
| 轴向承载力 | S_σ · S_L² |
| 抗弯刚度 | S_E · S_L⁴ |
| 抗弯承载力 | S_σ · S_L³ |

其中 S_L = 长度相似系数，S_E = 弹模相似系数，S_σ = 强度相似比
