使用说明

1. 系统简介

本系统基于相似理论，对矩形钢管截面进行缩尺模型优化设计。所有计算在浏览器本地完成，无需安装任何软件，打开网址即可使用。

2. 单工况设计

  1. 在左侧输入原型截面的长度 a、宽度 b、壁厚 t（mm）。
  2. 调整权重，控制各性能指标的优先级。
  3. 在中栏确认设计参数（相似系数、材料参数等）。
  4. 点击「优化计算」或开启「实时自动计算」，即可在右栏查看结果。

3. 批量计算

切换到「批量计算」页面，以 JSON 数组格式提交多组工况，系统将一次性返回所有结果。最多 50 组。

4. 导出功能

  • 下载报告：导出完整的 TXT 设计报告。
  • 导出 CSV：导出性能指标对比表格，可直接用 Excel 打开。

5. 优化算法

采用两阶段网格搜索算法：第一阶段粗搜确定可行域，第二阶段在最优解附近精细搜索。目标函数为各性能指标误差的加权平均值。

6. 相似理论公式

轴向刚度相似比 = SE · SL²

轴向承载力相似比 = Sσ · SL²

抗弯刚度相似比 = SE · SL⁴

抗弯承载力相似比 = Sσ · SL³

SL = 长度相似系数（模型/原型）
SE = 弹性模量相似系数（模型弹模/原型弹模）
Sσ = 强度相似比（模型强度/原型强度，由输入自动计算）

7. 技术说明

• 所有计算在浏览器端完成，数据不上传到任何服务器
• 方案历史保存在浏览器 localStorage 中
• 支持桌面端和移动端访问
